# GitHub Pages (JP/EN) - Personal

## Quick deploy
1) Create a repo named: `YOUR_GITHUB.github.io`
2) Upload everything in this bundle to the repo root (keep `assets/` and `en/`)
3) GitHub → Settings → Pages → Deploy from branch → `main` / root
4) Open:
- JP/EN toggle page: `https://YOUR_GITHUB.github.io/`
- English direct: `https://YOUR_GITHUB.github.io/en/`

## Update links
Edit these placeholders in:
- `index.html`
- `en/index.html`

Replace:
- `https://github.com/YOUR_GITHUB`
- `https://www.linkedin.com/in/YOUR_LINKEDIN`
